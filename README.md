# liri-bot

## Getting Started
        Clone down repo.
        Run command 'npm install' in Terminal or GitBash
        Run command 'node liri.js' or one of the commands below.


## What Each Command Does
### node liri.js spotify-this-song <song name>
        Shows the following information about the song in terminal/bash window.

        1. The album that the song is from
        2. Artist(s)
        3. The song's name
        4. A preview link of the song from Spotify

<img width="943" alt="song" src="https://user-images.githubusercontent.com/43390686/55045013-c702e580-5012-11e9-9ca3-0bc202d73598.png">
        

### node liri.js movie-this <movie name>
        Shows the following information in terminal/bash.

        1. Title of the movie.
        2. Year the movie came out.
        3. IMDB Rating of the movie.
        4. Rotten Tomatoes Rating.
        4. Country where the movie was produced.
        5. Language of the movie.
        6. Plot of the movie.
        7. Actors in the movie.
        

<img width="1035" alt="movie" src="https://user-images.githubusercontent.com/43390686/55039163-821e8500-4ff9-11e9-967f-722706eb234f.png">

### node liri.js concert <artist/band name>
        Shows the following information in terminal/bash.
        1. Venue Name.
